# Assets Directory

Place your images, icons, and other assets here.

## Suggested Assets to Add

- `avatar.png` - Profile avatar image
- `logo.svg` - Site logo
- `og-image.jpg` - OpenGraph image for social sharing

## Migrating from Astro Site

The original Astro site had these assets that can be copied:
- `src/assets/branding/profile.png` - Profile image
- `src/assets/branding/logo-favicon.svg` - Favicon/logo
- `src/assets/branding/og-image.jpg` - OG image

Copy these files from the `main` branch if needed.
